# ViniloStore

Proyecto demo: tienda online de vinilos creada con Vite + React.

Instrucciones rápidas:

1. Instala dependencias:

```bash
npm install
```

2. Ejecuta en desarrollo:

```bash
npm run dev
```

Coloca tus imágenes en `public/imagenes` y enlázalas en los productos dentro de `src/App.jsx`.
